# Тестое задание на Knockout

Группы списков документов с возможностью сортировки с помощью `drag and drop`.

Зависимости:

- jQuery^3.7.0
- jQueryUI^1.13.2
- knockout^3.5.1
- knockout-sortable^1.2.2

## Установка

```bash
# yarn
yarn install

# npm
npm install

# pnpm
pnpm install
```

## Запуск сервера

```bash
gulp serve
```

## Сборка проекта

```bash
gulp build
```
